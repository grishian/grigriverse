Welkom

Hier op GriGriVerse's server kan je je eigen eiland in de lucht bouwen. Je start met maar één kist met spullen!
Ga slim om met wat je hebt en bouw je eiland uit tot de grootste van de server!

Skyblock

Skyblock, de gamemode op onze server, is een populaire gamemode waar je start op een klein eiland in de lucht. Je moet gebruik maken van heel je eiland. De kist waarmee je start, maar ook de blokken waaruit je eiland zijn opgebouwd zijn essentieel!
Wil je meer leren over skyblock? Ga dan naar de Minecraft Wiki.

Server

Voorlopig hebben we enkel Skyblock beschikbaar. Uitbreiden naar meerdere gamemodes zal enkel gedaan worden als er meer spelers zijn.
Als je een voorstel of een vraag hebt kan je altijd via het minecraft command '/mail send Grishian mijnTekst' (enkel op de server) de admin bereiken. De website functie 'contact' is onder constructie.
