Server eerste bericht

Als je de server joint wordt je begroet met een berichtje. Als je op de blauwe link klikt kan je onze website ook bezoeken!

Spawn command

Typ /spawn . Na 3 seconden wordt je naar de spawn geteleporteerd. In die 3 seconden wachten mag je wel niet rondlopen of de teleportatie wordt geannuleerd.

Help command

Typ /help . Je krijgt een lijst met standaard commando's. Typ /help 2 , /help 3 ,... om meer commando's te zien. Je kan ook soms informatie zien over een command als je er 'help' voor zet. Voorbeeld /help mail . Laat je zien hoe je het /mail command moet gebruiken.

Kits command

Doe /kits . Om al je kits te zien. Doe bijvoorbeeld '/kit beginner' om de beginner kit te krijgen. Merk wel op dat je nu kit zonder de 's' schrijft.

Warps command

Doe '/warps' om een lijst met warps te zien. Net zoals bij kits schrijf je '/warp spawn' zonder 's'.

Island commando's

Doe '/is help' om al de commando's te zien die bij skyblock horen. Ook hier kan je '/is help 2', '/is help 3', ... doen.

'/is' command

Doe '/is' om het skyblock menu te zien. Hier kan je allerlei handige opties aanpassen. Zweef over de items om meer informatie te krijgen.

Island upgrades

Doe '/is upgrades' om je eiland te upgraden. Je zal wel een bepaalde hoeveelheid geld en crystals moeten sparen.

De shop

Doe '/is shop' om allerlei items te kopen.

Je eigen eiland

Doe '/is go' of '/is home' om naar je eigen eiland te teleporteren.
